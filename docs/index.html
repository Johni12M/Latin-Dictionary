What you get

Everything a Latin student needs

⚡

Instant Lookup

Search any Latin word and get full grammatical information in under a second — declensions, conjugations, and meanings.

📖

Full Grammar Info

See the word's head form, declension or conjugation class, all forms highlighted, and German meanings.

📜

Search History

Your last 50 searches are stored locally so you can quickly revisit words you've looked up before.

⚙️

Offline Cache

Results are cached on your device after the first lookup — repeated searches are instant even without internet.

🎨

Beautiful Dark UI

Clean, distraction-free interface with a deep indigo theme and animated search indicator built with Flet.

📦

Single File Exe

No Python installation required. One 89 MB executable or a simple installer — runs on any Windows 10/11 PC.
